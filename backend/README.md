# Backend Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Set your Julep API key and environment in `.env`:
   ```env
   JULEP_API_KEY=your_julep_api_key_here
   JULEP_ENVIRONMENT=production
   ```
3. Start the server:
   ```bash
   npm start
   ```

The server will run on port 3001 by default.
